# opl
opl is a shell command that will search your entire computer for a file or directory.

## Install

1. Install [`python`](https://python.org)
2. Run `install.sh`
3. Run `opl opl`
4. If returns text (not error), it is installed, otherwise you can [submit an issue](https://github.com/TheRealPenguin12/opl/issues).
